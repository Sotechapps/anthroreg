What the tool does

It accepts paired measurements of hand length and stature, then calculates the regression constant, regression coefficient, correlation coefficient, R squared, and standard error of estimate.

Why it matters

Anthropometric regression can support biological profiling by estimating stature from measurable body segments when direct stature measurement is unavailable.

How to use it

Choose gender and age, build an equation from data, save it, then use the stature estimation page to estimate stature from a hand length value.
